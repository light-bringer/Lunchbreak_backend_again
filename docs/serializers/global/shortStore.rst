ShortStore
==========

Fields
------
    - **id** (*int*) - Store ID
    - **name** (*String*) - Name
    - **city** (*String*) - City
    - **street** (*String*) - Street
    - **latitude** (*decimal*) - Latitude coordinate
    - **longitude** (*decimal*) - Longitude coordinate
    - **categories** (:doc:`/serializers/global/storeCategory` list) - Store categories
    - **heartsCount** (*int*) - Amount of hearts.
