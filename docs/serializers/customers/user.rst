User
====

* Fields
    - **id** (*int*) - User ID
    - **name** (*String*) - Name, not required to write to if it's already in the database
    - **phone** (*phonenumber*) - Phone number
    - **pin** (*String*) - Pin, not required
    - **device** (*String*) - Device, not required

* Read only fields
    - id

* Write only fields
    - phone
    - pin
    - device
